# 📂 Shared Folders and Network Printer

## 📌 Overview

Shared resources such as folders and printers allow multiple users in a network to access files and printing services from a central server.

This improves collaboration, reduces duplication, and enables efficient resource management in an organisation.

---

## 📌 Purpose

- Share files across multiple users
- Centralise data storage
- Provide network printing
- Improve collaboration and productivity

---

## 📁 Shared Folder Configuration

### 🔹 Setup Steps

1. Create a folder on the server
2. Right-click → **Properties**
3. Go to **Sharing** tab
4. Click **Advanced Sharing**
5. Enable **Share this folder**
6. Assign a share name
7. Configure permissions:
   - Read
   - Write
   - Full Control (based on requirements)
