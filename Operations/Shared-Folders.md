# 📂 Shared Folder Configuration

## 📌 Overview

Shared folders allow multiple users on a network to access files from a central server location.

This enables collaboration, centralised storage, and easier file management within an organisation.

---

## 📌 Purpose

- Share files across users
- Centralise data storage
- Control access using permissions
- Improve collaboration

---

## 📁 Shared Folder Setup

### 🔹 Steps

1. Create a folder on the server  
2. Right-click → **Properties**  
3. Go to **Sharing** tab  
4. Click **Advanced Sharing**  
5. Enable **Share this folder**  
6. Assign a share name  
7. Click **Permissions** and configure access

## 🔐 Permissions Configuration

| Permission | Description |
|-----------|------------|
| Read | Users can view files |
| Change | Users can modify files |
| Full Control | Users have complete access |

---

## 📌 Key Point

Permissions should follow the **least privilege principle**, giving users only the access they need.
