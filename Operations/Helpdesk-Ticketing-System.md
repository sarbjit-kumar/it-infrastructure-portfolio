# 🎫 Helpdesk & Ticketing System

## 📌 What is a Helpdesk?
A helpdesk is a support system that assists users in resolving technical issues related to IT services such as network, hardware, and software.

It acts as the first point of contact between users and IT support teams.

---

## 📌 What is a Ticketing System?
A ticketing system is used to track, manage, and resolve user issues.

Each issue is recorded as a “ticket” which includes:
- User details  
- Problem description  
- Status  
- Solution  
- Resolution time  

---

## 📌 Ticket Lifecycle
Open → In Progress → Escalated → Resolved → Closed

---

## 📌 Key Responsibilities
- Communicate with users professionally  
- Understand the issue clearly  
- Provide troubleshooting steps  
- Escalate when required  
- Follow up with users  
- Close ticket after resolution  

---

## 🧾 Example Scenario 1 – Wi-Fi Issue

**Problem:**  
User unable to connect devices to Wi-Fi.

**Action Taken:**
- Asked diagnostic questions  
- Identified Wi-Fi signal issue  
- Guided user to restart router  
- Escalated to technician  

**Outcome:**  
Issue resolved and devices reconnected successfully.

---

## 🧾 Example Scenario 2 – Printer Issue

**Problem:**  
User unable to print wirelessly.

**Action Taken:**
- Tested printer via USB  
- Identified network issue  
- Reconfigured Wi-Fi settings  
- Escalated if needed  

**Outcome:**  
Printer connected and working on all devices.

---

## 🧾 Example Scenario 3 – RAM Issue

**Problem:**  
System shutting down after RAM installation.

**Action Taken:**
- Checked compatibility  
- Tested RAM individually  
- Identified faulty RAM  

**Outcome:**  
Faulty RAM replaced and system working.
