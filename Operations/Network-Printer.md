# 🖨️ Network Printer Configuration

## 📌 Overview

A network printer allows multiple users to print documents through a shared printer connected to a server.

This centralised setup improves efficiency and reduces the need for individual printers.

---

## 📌 Purpose

- Provide shared printing services  
- Reduce hardware cost  
- Centralise printer management  
- Allow multiple users to print over network  

---

## 🖨️ Printer Setup on Server

### 🔹 Steps

1. Connect or install printer on the server  
2. Open **Devices and Printers**  
3. Right-click printer → **Printer Properties**  
4. Go to **Sharing** tab  
5. Enable **Share this printer**  
6. Assign a printer share name

## 🔗 Client Connection to Printer

### 🔹 Steps

1. Open **Devices and Printers**  
2. Click **Add Printer**  
3. Select the network/shared printer  
4. Install required drivers
