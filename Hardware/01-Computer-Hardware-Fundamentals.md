# 🖥️ Computer Hardware Fundamentals

## 📌 Overview

This document explains the main components of a computer system, including their purpose, types, and importance in system performance.

Understanding these components is essential for building, upgrading, and troubleshooting computers.

---

## 🧩 Motherboard

### 📌 Purpose of Motherboard

A motherboard is a printed circuit board (PCB) that connects all components of a computer system in one place.

It allows communication between:
- CPU  
- RAM  
- Storage devices  
- Expansion cards  

👉 It is also known as:
- Mainboard  
- System board  
- Logic board  

---

### 📌 Form Factors

The layout and size of a motherboard are called its form factor.

#### Common Types:
- **ATX** → Standard size with more slots and expansion options  
- **EATX** → Larger, used for high-performance systems  
- **Micro-ATX** → Smaller with fewer expansion slots  
- **Mini-ITX** → Compact, low power consumption  

---

### 📌 Key Components

- CPU socket  
- RAM slots  
- BIOS/UEFI  
- CMOS battery  
- Expansion slots (PCIe)  
- SATA connectors  
- I/O ports  

---

## 🧠 Central Processing Unit (CPU)

### 📌 Purpose of CPU

The CPU (Central Processing Unit) is the main component of a computer and is known as the **brain of the system**.

It:
- Fetches instructions  
- Decodes instructions  
- Executes operations  
- Performs calculations  

👉 A faster CPU improves system performance.

---

### 📌 Components of CPU

- **Control Unit** → Controls all operations  
- **Memory Unit** → Stores instructions temporarily  
- **ALU (Arithmetic Logic Unit)** → Performs calculations  

---

### 📊 CPU Comparison

| Feature | Intel i7-13700KF | Intel i9-13900K |
|--------|------------------|----------------|
| Cores | 16 | 24 |
| Threads | 24 | 32 |
| Max Speed | 5.4 GHz | 5.8 GHz |
| Cache | 30 MB | 36 MB |
| Power | 125W | 125W |

👉 The i9 provides higher performance due to more cores and higher speed.

---

## 💾 Random Access Memory (RAM)

### 📌 Purpose of RAM

RAM is temporary memory used by the CPU to store data while processing tasks.

👉 It is:
- Fast  
- Volatile (data is lost when power is off)  

---

### 📌 Importance of RAM

- Faster data access  
- Improves performance  
- Supports multitasking  

---

### 📌 Types of RAM

#### 🔹 SRAM
- Faster  
- More expensive  
- Used in cache memory  

#### 🔹 DRAM
- Slower than SRAM  
- Cheaper  
- Used as main memory  

---

### 📊 RAM Comparison

| Feature | DDR4 | DDR5 |
|--------|------|------|
| Speed | 3200 MHz | 6000 MHz |
| Performance | Good | Higher |
| Cost | Lower | Higher |

---

### 📌 Form Factors

- **DIMM** → Desktop computers  
- **SODIMM** → Laptops  

---

## 💽 Fixed Storage Devices

### 📌 Purpose

Fixed storage devices store data permanently, even when the computer is turned off.

---

### 📌 Types of Storage

#### 🔹 HDD (Hard Disk Drive)
- Uses spinning disks  
- Large capacity  
- Slower performance  

---

#### 🔹 SSD (Solid State Drive)
- No moving parts  
- Faster than HDD  
- More reliable  

---

#### 🔹 NVMe Drive
- Uses PCIe interface  
- Much faster than SSD and HDD  
- Used for high-performance systems  

---

### 📌 Key Difference

👉 Speed comparison:

HDD < SSD < NVMe  

---

## 🔌 Power Supply Unit (PSU)

### 📌 Purpose of PSU

The PSU supplies power to all computer components.

It:
- Converts AC to DC  
- Distributes power to all components  

---

### 📌 Power Rating

Measured in Watts (W)

👉 Higher wattage is required for:
- Gaming PCs  
- High-performance systems  

---

### 📌 Types of PSU

- **Non-Modular** → Fixed cables  
- **Semi-Modular** → Some removable cables  
- **Fully Modular** → All cables removable  

---

### 📌 Form Factors

- ATX  
- SFX  
- Flex ATX  
- TFX
