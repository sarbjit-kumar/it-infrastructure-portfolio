# 💾 iSCSI Storage Configuration

## 📌 Overview
This project demonstrates configuring iSCSI storage to provide remote disk access over a network.

---

## ⚙️ What I Implemented
- Configured iSCSI target on file server
- Connected using iSCSI initiator
- Created and mounted disk on client

---

## 🧠 Key Concepts
- Block-level storage
- Network-based disk access
- Target vs Initiator

---

## ✅ Validation
- Disk visible in Disk Management
- Storage accessible and usable

---

## 💼 Real-World Use
Used in enterprise storage systems and virtual environments for centralized storage solutions.
