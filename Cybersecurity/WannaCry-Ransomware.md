# 🔐 WannaCry Ransomware Attack (Case Study)

## 📌 Overview
This is a cybersecurity case study on the WannaCry ransomware attack, originally researched and documented as part of my coursework.

---

## 📌 What is WannaCry?
WannaCry is a ransomware attack that spread globally in May 2017, encrypting files on infected systems and demanding payment in Bitcoin.

---

## ⚙️ How it Worked
1. Exploited SMB vulnerability (EternalBlue)  
2. Spread automatically across networks  
3. Encrypted user files  
4. Displayed ransom message  

---

## 💥 Impact / Damage
- 230,000+ systems infected  
- 150+ countries affected  
- Estimated $4 billion damage  
- Affected hospitals, businesses, governments  

---

## 🧠 Key Technical Insight
- Used SMB protocol vulnerability  
- Spread like a worm  
- Targeted unpatched systems  

---

## 🛡️ Prevention / Protection
- Keep systems updated (WSUS)  
- Maintain regular backups  
- Use antivirus protection  
- Avoid outdated operating systems  
- Apply security policies  

---

## 🔗 Integration with My Labs
- WSUS → patch vulnerabilities  
- Backup → recover encrypted data  
- GPO → enforce security settings  
- Monitoring → detect unusual activity  

---

## 💼 Real-world Relevance
This attack highlights the importance of patch management, backup strategies, and network security in enterprise environments.
