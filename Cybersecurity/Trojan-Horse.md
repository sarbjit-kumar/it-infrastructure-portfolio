# 🐴 Trojan Horse Attack

## 📌 What is a Trojan Horse?
A Trojan Horse is a type of malware that disguises itself as legitimate or useful software to trick users into installing it.

---

## ⚙️ How it Works
1. Attacker creates malicious software disguised as legitimate  
2. User downloads or installs it  
3. Malware activates after installation  
4. Attacker gains access to the system  

---

## 💥 Impact / Damage
- Unauthorized system access  
- Data theft  
- Installation of additional malware  
- Full system compromise  

---

## 🛡️ Prevention / Protection
- Download software from trusted sources only  
- Avoid pirated/cracked software  
- Use antivirus protection  
- Do not open unknown attachments  
- Keep systems updated  

---

## 🧠 Key Concepts
- Trojan = disguised malware  
- Requires user action (installation)  
- Does not self-replicate  

---

## 🔄 Trojan vs Worm vs Virus
- Trojan → fake software  
- Worm → spreads automatically  
- Virus → attaches to files  

---

## 🔗 Integration with IT Systems
- Security policies restrict installations  
- Antivirus detects threats  
- User awareness prevents attacks  

---

## 💼 Real-world Relevance
Trojan attacks are common in fake software downloads and phishing attachments, leading to system compromise.

---
