# 💻 Visual Studio Fundamentals & Programming Concepts

## 📌 What is Software Development Life Cycle (SDLC)?

The Software Development Life Cycle (SDLC) is a structured process used to design, develop, test, and deploy software applications.

### Key Phases:
- Planning
- Design
- Development
- Testing
- Deployment
- Maintenance

👉 SDLC helps developers move software through organised stages.

---

## 🎯 Importance of SDLC

- Provides a structured approach to development
- Reduces errors and risks
- Improves planning and testing
- Helps teams work more effectively

---

## 📌 Planning Phase for Windows Forms

Before creating a Windows Form, planning is important.

### Planning Includes:
- Identifying user requirements
- Deciding form layout
- Selecting controls
- Planning validations
- Designing user flow

---

## 📌 Programming Principles

### 1. KISS
Keep code simple and easy to understand.

### 2. DRY
Avoid repeating the same code multiple times.

### 3. YAGNI
Write only the code that is needed now.

### 4. Separation of Concerns
Divide the program into smaller independent parts.

---

## 📌 Procedural vs Object-Oriented Programming

### 🔹 Procedural Programming
- Based on functions
- Step-by-step approach
- Focuses on tasks

### 🔹 Object-Oriented Programming
- Based on objects and classes
- Models real-world scenarios
- Better for larger applications

---

## 📌 Visual Studio

Visual Studio is an Integrated Development Environment (IDE) used to create different applications.

### Examples:
- Windows applications
- Web applications
- Mobile apps
- Games
- Database applications

---

## 📌 Visual Basic Controls

Common controls used in forms:

- **TextBox** → User input
- **Label** → Display field names
- **Button** → Perform actions
- **ComboBox** → Select options
- **DateTimePicker** → Select date
- **CheckBox** → Multiple selections

---

## 📌 What are Validations?

Validations ensure that user-entered data is correct and in the required format.

### Examples:
- Empty field validation
- Email format validation
- Phone number validation
- Date validation

---
