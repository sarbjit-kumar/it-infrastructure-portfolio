# 🌐 Portfolio Website Project

## 📌 Overview

This project involved creating a personal portfolio website using HTML and CSS.

The website was developed to present personal information, qualifications, and work experience in a structured and professional format.

---

## 🎯 Purpose

- Build a personal portfolio website  
- Apply HTML and CSS knowledge  
- Practice web design and layout  
- Present information professionally  

---

## ⚙️ Tools and Environment

- Visual Studio Code  
- Web browser (Chrome / Edge)  

---

## 🗂️ Project Setup

### 🔹 Steps

1. Installed Visual Studio Code  
2. Created a folder named **Portfolio**  
3. Created main file:
   - `index.html`
