# 🌐 Web Design Fundamentals

## 📌 Overview

Web design focuses on creating websites that are easy to use, visually appealing, and accessible.

---

## 🎯 Purpose

- Understand user interaction  
- Learn design principles  
- Improve user experience  

---

## 🖥️ Human Computer Interaction (HCI)

HCI focuses on how users interact with systems and improving usability.

### Why it matters:
- Improves usability  
- Reduces errors  
- Enhances experience  

---

## ⭐ Norman’s Principles

| Principle | Meaning |
|----------|--------|
| Visibility | Options are visible |
| Feedback | System responds |
| Affordance | Easy to understand usage |
| Mapping | Controls match results |
| Constraints | Prevent errors |
| Consistency | Same actions = same result |

---

## ♿ Accessibility

- Alt text for images  
- Keyboard access  
- Clear structure  
- Good contrast  
- Captions for media  

👉 Goal: usable by everyone  

---

## 🌐 Good vs Bad Design

### Good:
- Simple  
- Fast  
- Easy navigation  

### Bad:
- Cluttered  
- Slow  
- Confusing  

---

## 🎨 Media Design Principles

| Principle | Purpose |
|----------|--------|
| Balance | Visual stability |
| Emphasis | Highlight key info |
| Repetition | Consistency |

---

## 🔥 Key Learning

Good design focuses on **user experience, not just looks**.
