# 🖥️ VM Creation using PowerShell (Hyper-V)

## 📌 Overview
This project demonstrates automated virtual machine creation using PowerShell in Hyper-V.

---

## ⚙️ What I Implemented
- Created VM using PowerShell
- Attached virtual hard disk (VHD)
- Connected network switch
- Mounted ISO for OS installation

## 🧠 Key Concepts
- Infrastructure automation
- Hyper-V management using PowerShell
- Repeatable deployment
