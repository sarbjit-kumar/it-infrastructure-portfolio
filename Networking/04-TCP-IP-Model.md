# 🌐 TCP/IP Model (Transmission Control Protocol / Internet Protocol)

## 📌 Overview

The TCP/IP model is a practical networking model used in real-world networks and the internet. It defines how data is transmitted between devices across networks.

Unlike the OSI model, TCP/IP has fewer layers and focuses on real communication processes.

---

## 🧱 Layers of TCP/IP Model

The TCP/IP model consists of five layers:

1. Application Layer  
2. Transport Layer  
3. Internet Layer  
4. Data Link Layer  
5. Physical Layer  

---

## 5️⃣ Application Layer

### 📌 Function

- Provides services to end users  
- Combines OSI Application, Presentation, and Session layers  

### 📌 Examples

- HTTP (web browsing)  
- FTP (file transfer)  
- SMTP (email)  
- DNS (domain name system)  

---

## 4️⃣ Transport Layer

### 📌 Function

- Ensures data is delivered correctly  
- Breaks data into segments  
- Handles error detection and recovery  

### 📌 Protocols

#### TCP (Transmission Control Protocol)

- Reliable and connection-oriented  
- Ensures data is delivered correctly  
- Used for important data  

#### UDP (User Datagram Protocol)

- Faster but less reliable  
- No error checking  
- Used for streaming and real-time applications  

---

## 3️⃣ Internet Layer

### 📌 Function

- Responsible for logical addressing and routing  
- Determines the path for data transmission  

### 📌 Protocols

- IP (Internet Protocol)  
- ICMP (used for ping)  

---

## 2️⃣ Data Link Layer

### 📌 Function

- Handles communication between devices on the same network  
- Uses MAC addresses  
- Detects errors  

---

## 1️⃣ Physical Layer

### 📌 Function

- Transmits data as signals through cables or wireless  
- Includes hardware components  

---

## 📊 TCP/IP Layer Summary

| Layer | Function | Example |
|------|---------|--------|
| Application | User services | HTTP, FTP, DNS |
| Transport | Reliable delivery | TCP, UDP |
| Internet | Routing | IP |
| Data Link | Local communication | MAC |
| Physical | Hardware | Cables |

---

## 🔄 Difference Between OSI and TCP/IP

| Feature | OSI Model | TCP/IP Model |
|--------|----------|-------------|
| Layers | 7 | 5 |
| Type | Theoretical | Practical |
| Usage | Learning & design | Real-world networks |
| Structure | More detailed | Simpler |

---

## 📌 Importance of TCP/IP Model

The TCP/IP model is important because:

- It is used in real-world networking  
- Forms the basis of the internet  
- Helps in understanding communication between devices  
- Supports troubleshooting and network design  

---

## 📌 Real-world Relevance

TCP/IP model is used in:

- Cloud computing

- IT support and networking
