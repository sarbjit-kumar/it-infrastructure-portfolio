# 🧱 OSI Model (Open Systems Interconnection)

## 📌 Overview

The OSI (Open Systems Interconnection) model is a conceptual framework used to understand how data is transmitted over a network. It divides communication into seven layers, each with a specific function.

The OSI model helps in designing, understanding, and troubleshooting network systems.

---

## 🧱 7 Layers of OSI Model

---

## 7️⃣ Application Layer

### 📌 Function

- Provides network services to end users  
- Acts as the interface between user applications and the network  

### 📌 Examples

- Web browsers  
- Email clients  

---

## 6️⃣ Presentation Layer

### 📌 Function

- Formats and translates data  
- Handles encryption and decryption  
- Compresses data if required  

---

## 5️⃣ Session Layer

### 📌 Function

- Establishes, manages, and terminates communication sessions  
- Maintains connection between devices  

---

## 4️⃣ Transport Layer

### 📌 Function

- Ensures reliable data delivery  
- Breaks data into segments  
- Handles error detection and correction  

### 📌 Protocols

- TCP (reliable)  
- UDP (fast, less reliable)  

---

## 3️⃣ Network Layer

### 📌 Function

- Handles routing of data  
- Determines best path for data transmission  
- Uses IP addresses  

---

## 2️⃣ Data Link Layer

### 📌 Function

- Provides node-to-node communication  
- Handles error detection  
- Uses MAC addresses  

---

## 1️⃣ Physical Layer

### 📌 Function

- Transmits raw bits over physical media  
- Includes cables, switches, and hardware  

---

## 📊 OSI Layer Summary

| Layer | Name | Function |
|------|------|---------|
| 7 | Application | User interaction |
| 6 | Presentation | Data formatting & encryption |
| 5 | Session | Session management |
| 4 | Transport | Reliable delivery |
| 3 | Network | Routing (IP) |
| 2 | Data Link | MAC & error detection |
| 1 | Physical | Hardware transmission |

---

## 📌 Importance of OSI Model

The OSI model is important because it:

- Simplifies network design  
- Helps in troubleshooting issues  
- Provides a standard communication structure  
- Allows different systems to communicate  

---

## 📌 Real-world Relevance

OSI model is used in:

- Network troubleshooting  
- System design  
- IT support roles  
- Network engineering  

👉 It helps identify where problems occur in a network.

---

## 🔥 Important Line

The OSI model breaks network communication into layers, making it easier to understand, design, and troubleshoot networks.
