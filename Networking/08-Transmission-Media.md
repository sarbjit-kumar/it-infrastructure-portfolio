# 🔌 Transmission Media

## 📌 Overview

Transmission media refers to the physical path or medium through which data is transmitted between devices in a network.

It plays a critical role in network performance, speed, and reliability.

Transmission media is mainly divided into:

- Wired (Guided) Media  
- Wireless (Unguided) Media  

---

## 🧵 Wired (Guided) Media

Wired media uses physical cables to transmit data.

---

## 🔹 Twisted Pair Cable (UTP / STP)

### 📌 Definition

Twisted pair cables consist of pairs of wires twisted together to reduce interference.

### 📌 Types

- **UTP (Unshielded Twisted Pair)** → Common and low cost  
- **STP (Shielded Twisted Pair)** → Better protection from interference  

### 📌 Advantages

- Low cost  
- Easy to install  
- Widely used  

### 📌 Disadvantages

- Limited distance  
- Susceptible to interference (especially UTP)  

### 📌 Use Case

- Ethernet networks  
- Office LANs  

---

## 🔹 Coaxial Cable

### 📌 Definition

Coaxial cable has a central conductor surrounded by insulation and shielding.

### 📌 Advantages

- Better shielding than twisted pair  
- Less interference  

### 📌 Disadvantages

- Bulkier and less flexible  
- More expensive than UTP  

### 📌 Use Case

- Cable TV  
- Older network systems  

---

## 🔹 Fibre Optic Cable

### 📌 Definition

Fibre optic cables use light signals to transmit data at very high speeds.

### 📌 Advantages

- Very high speed  
- Long-distance communication  
- No electromagnetic interference  

### 📌 Disadvantages

- Expensive  
- Difficult to install and maintain  

### 📌 Use Case

- Internet backbone  
- Data centres  
- Long-distance communication  

---

## 📡 Wireless (Unguided) Media

Wireless media transmits data without physical cables.

---

## 🔹 Wi-Fi

### 📌 Definition

Wi-Fi uses radio waves to connect devices wirelessly.

### 📌 Advantages

- Mobility and flexibility  
- Easy to set up  

### 📌 Disadvantages

- Signal interference  
- Limited range  

### 📌 Use Case

- Home and office networks  

---

## 🔹 Bluetooth

### 📌 Definition

Bluetooth is used for short-range communication between devices.

### 📌 Advantages

- Low power consumption  
- Easy device pairing  

### 📌 Disadvantages

- Very short range  
- Lower data speed  

### 📌 Use Case

- Headphones  
- Mobile devices  

---

## 🔹 Satellite Communication

### 📌 Definition

Satellite communication uses satellites to transmit data over large distances.

### 📌 Advantages

- Wide coverage (global)  

### 📌 Disadvantages

- High latency  
- Expensive  

### 📌 Use Case

- Remote areas  
- Global communication  

---

## 📊 Wired vs Wireless Comparison

| Feature | Wired | Wireless |
|--------|------|----------|
| Speed | High | Medium |
| Reliability | High | Lower |
| Mobility | Low | High |
| Installation | Complex | Easy |

---

## 📌 Key Learning

- Wired media provides better speed and reliability  
- Wireless media provides flexibility and mobility  
- Choice depends on network requirements  

---

## 📌 Real-world Relevance

Transmission media is used in:

- Office networks  
- Data centres  
- Home networks  
- Telecommunications  

👉 It directly affects network performance and efficiency.

---

## 🔥 Important Line

Transmission media determines how data is physically transmitted and directly impacts network speed, reliability, and coverage.

---

## 🧠 Simple Understanding

Wired = fast + stable  
Wireless = flexible + mobile
