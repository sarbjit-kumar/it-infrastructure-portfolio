# ⚡ Active Directory Automation (PowerShell)

## 📌 Overview
This project demonstrates automation of Active Directory tasks using PowerShell scripts.

---

## ⚙️ What I Implemented
- Created Organizational Units (OUs)
- Created security groups
- Bulk user creation using CSV file
- Automated user assignment to groups

---

## 🧠 Key Concepts
- CSV-based data import
- PowerShell scripting
- Loop automation
- Scalable user provisioning

---

## ✅ Validation
- Users successfully created in ADUC
- Groups assigned correctly
- No script errors

---

## 💼 Real-World Use
Used in enterprise environments to automate onboarding and manage large numbers of users efficiently.
